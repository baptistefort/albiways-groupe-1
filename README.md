# VISION IA x Abilways - Challenge chatbot IA

Présentation interactive en HTML/CSS/JS pour le challenge Abilways.

Ouvrir `index.html` dans un navigateur. La page fonctionne sans serveur local.

Le fil rouge : construire un chatbot IA texte pour le site VISION IA, capable de comprendre l'agence, de comprendre l'entreprise du prospect, puis de proposer une première piste d'automatisation.

Chaque bloc contient un chapitre tutoriel détaillé, 4 images, 3 schémas, des prompts et des points de contrôle.
